# R.R.R.
La función de este vos es de informarte sobre qué es lo que puedes hacer con cada tipo de desecho para ayudar a reducir la cantidad de desechos que produces diariamente.

Aquí estan todas las funciones del bot:

## -res_sort :
Describe lo que puedes hacer con cada uno de los residuos registrados.

## -org_link :
Proporciona links de organizaciones centradas en el reciclaje y la sostenibilidad dentro de Latinoamérica. Si usted tiene otros links, escríbalos dentro de la lista de links.
